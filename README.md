# 🚗 Commute Time Analytics System

## 📌 Overview
This project analyzes and compares employee commute times between home locations and two office options using Python and Pandas. It generates structured insights and exports results into a formatted Excel report.

---

## 🎯 Problem Statement
Organizations often struggle to evaluate which office location is more convenient for employees. This project solves that by comparing commute times and highlighting differences between two office routes.

---

## ⚙️ Features
- Load employee dataset from CSV
- Clean and structure data using Pandas
- Simulate/estimate commute times
- Compare Office 1 vs Office 2 travel time
- Calculate difference (Delta)
- Export results to Excel
- Apply conditional formatting (Green/Red)

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- OpenPyXL
- Jupyter Notebook
